# 🍽️ FreshBite Foods Co. - Sales Dashboard

## 📌 About This Project

This project is a simple interactive **Sales Dashboard** built using Python, Dash, and Plotly.
It shows how a fictional company called *FreshBite Foods Co.* is performing in terms of sales across different cities in the Philippines.

The goal of this project is to practice data visualization and create a dashboard that helps users understand sales trends easily.



## 🏢 About the Company

FreshBite Foods Co. is a made-up food business that sells:

* Fast food (like burgers, pizza, fries)
* Drinks (like milk tea and coffee)

They operate in different cities such as:

* Iloilo City
* Cebu City
* Manila
* Davao City



## 🎯 What the Dashboard Can Do

* Filter sales by **city** and **category**
* Show total **revenue**, **quantity sold**, and **average price**
* Display:

  * Bar chart (revenue by product)
  * Line chart (sales over time)
  * Pie chart (category distribution)

## ⚙️ How to Install

1. Download or clone the project
2. Open your terminal or command prompt
3. Install the required libraries:

pip install -r requirements.txt



## ▶️ How to Run

After installing everything, run this command:

python app.py

Then open your browser and go to:

http://127.0.0.1:8050/



## 📊 About the Dataset

The dataset includes the following columns:

* Date – when the sale happened
* City – where the sale was made
* Product – what item was sold
* Category – type of product
* Price – price per item
* Quantity – number of items sold
* Revenue – total sales (price × quantity)
